# Job Portal MVC Application

A web-based job portal application built using Java, Spring Boot, and MVC architecture. The system allows users to manage job postings and perform CRUD operations.

## Features
- Add, update, delete job postings
- View all available jobs
- MVC-based architecture (Controller-Service-Repository)
- Database integration using MySQL

## Tech Stack
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL

## How to Run
1. Clone the repository
2. Configure database in application.properties
3. Run Spring Boot application
4. Access on localhost:8080
